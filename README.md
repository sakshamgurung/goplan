## Pre-Requisites

1. Install and start `mongodb`
2. In `client` folder, copy `.env.example` to `.env`
3. In `server` folder, copy `.env.example` to `.env`

## Install Dependencies

```bash
npm run setup
# OR
yarn setup
```

## Start app

```bash
npm run serve
# OR
yarn serve
```
